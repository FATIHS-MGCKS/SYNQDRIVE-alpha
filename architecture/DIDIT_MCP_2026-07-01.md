# Changes & Architektur — Didit MCP + Production Env (2026-07-01)

## Changes

- Added `.cursor/mcp.json` with hosted Didit MCP (`https://mcp.didit.me/mcp`).
- Added `docs/didit-setup.md` — MCP OAuth, webhook registration, workflow env, VPS sync.
- Added `backend/scripts/ops/sync-didit-env-to-vps.sh` — copies `DIDIT_*` block local → VPS.
- `didit.config.ts` reads `DIDIT_WORKFLOW_ID_*` from env (fallback to file placeholders).

## Architektur

- **Didit MCP** (OAuth user context) is for setup/ops: list workflows, manage webhooks, inspect sessions.
- **SynqDrive backend** uses application `DIDIT_API_KEY` + `DIDIT_WEBHOOK_SECRET` (server env, not MCP).
- **Webhook path unchanged:** `POST /api/v1/webhooks/didit` with `x-timestamp` + `x-signature-v2`.
- **Production env source:** `/opt/synqdrive/shared/backend.env` (synced from local `backend/.env`).
